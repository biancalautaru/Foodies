# Foodies

*Foodies* is a food delivery service system that manages customers, restaurants, drivers, and orders.

## Features

- **Customer Management**: Browse restaurants, manage cart, place orders, leave reviews
- **Restaurant Management**: Add menu items, manage availability and ratings
- **Order Management**: Track order status from placement to delivery
- **Driver Assignment**: Assign drivers to orders with vehicle types (scooter, car)
- **Review System**: Customers can rate and review restaurants after delivery
- **Order Cancellation**: Cancel orders with applicable fees based on status

## Getting Started

Run the `Main` class to see a complete demonstration of the system functionality.

## Project Structure

- `models/` - Core domain models (Customer, Driver, Restaurant, Order, etc.)
- `service/` - FoodDeliveryService for managing the platform operations
- `Main.java` - Demo application showcasing full workflow
